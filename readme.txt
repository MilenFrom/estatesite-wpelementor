Elementor widgets, dynamic tags, theme builder integration, and templates library for EstateSite.

== Description ==

Adds EstateSite-specific widgets, dynamic tags, and templates to Elementor. Requires the EstateSite Core plugin and Elementor (free is sufficient; theme-builder features require Pro).
