# EstateSite Elementor

Elementor widgets, dynamic tags, theme builder, and templates library for [EstateSite](https://estatesite.eu).

This plugin provides:
- **~140 ported Elementor widgets** across 6 categories:
  - `single-property/` (50 widgets — title, price, gallery, sections, etc.)
  - `single-agent/` (24 widgets)
  - `single-agency/` (25 widgets)
  - `single-post/` (8 widgets)
  - `header-footer/` (7 widgets — logo, menu, currency, language switchers)
  - `absorbed/` (12 widgets — listings, cards, favorites, team)
- **Theme builder** for 12 template types: Header, Footer, Before/After Header, Before/After Footer, Mega Menu, Custom Block, Single Listing, Single Agent, Single Agency, Single Post
- **Editor preview system** — Preview Settings panel + Apply & Preview, body-class injection, autosave-aware preview-target reads
- **Templates library** with 367 ready-to-insert Elementor templates
- **Dynamic tags**: property title, excerpt, content, agent phone
- **Custom Elementor controls**

## Dependencies

- **[EstateSite Core](https://github.com/MilenFrom/estatesite-wpcore)** (hard dependency) — install Core first
- **Elementor** (free) — required
- **Elementor Pro** — optional, only needed for advanced theme-builder features

## Install

1. Make sure EstateSite Core is installed and active.
2. Download the latest `.zip` from the [Releases](https://github.com/MilenFrom/estatesite-wpelementor/releases) page.
3. WordPress Admin → Plugins → Add New → Upload Plugin → choose the zip.
4. Activate.

Future updates appear automatically in Dashboard → Updates.

## Requirements

- WordPress 6.4+
- PHP 7.4+
- EstateSite Core (this package's hard dep)
- Elementor 3.x or 4.x
